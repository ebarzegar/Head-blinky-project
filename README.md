# Blinky Face (Java)
This project is a Java Swing application that draws an interactive face. The face reacts to mouse movement, changing its eyes when the cursor enters or exits the panel (blinking effect).

## Features
- Interactive GUI using Java Swing
- Mouse event handling (hover detection)
- Dynamic drawing with Graphics2D
- Smooth rendering with anti-aliasing

## Concepts Used
- Event-driven programming (MouseListener)
- Java Swing (JFrame, JPanel)
- Graphics and shapes (Ellipse2D, Line2D, Rectangle2D)
- Conditional rendering based on user interaction
